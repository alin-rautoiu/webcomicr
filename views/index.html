Usage

  • /getCorgi/:id
    Primești proprietățile corgiului cu Idul id,
  • /getAllCorgis/
    Primești toți corgii.
  • /getCorgiFace/:id
    Descarcă fața unui corgi.
  • /getRandomCorgi
    Primeste un caine random.

Structura caine

"dog" { "id" : "int",
"face" : "string",
"name" : "string",
"likes" : ["string"]
}
